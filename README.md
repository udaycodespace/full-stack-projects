# dev-projects

A collection of full stack development projects spanning frontend frameworks, backend systems, and complete full stack applications.

## Structure

### frontend
- `angular/` — Angular core concept implementations: components, templates, routing, services, API integration
- `html-css-js/` — Vanilla web projects including an interactive to-do app with drag-and-drop and LocalStorage
- `react/` — React projects including an internal operations frontend with RTK Query

### backend
- `java-springboot/` — Spring Boot projects: student management system and JWT authentication system
- `python/` — Python backend projects: API data integration and SQL client reporting
- `devops/` — Dockerized applications including a number guesser game

### fullstack
- `mern/` — Full MERN stack projects including LearnHub (e-learning platform) and CampusEventHub (team project)

## Stack
React · Angular · HTML · CSS · JavaScript · Java · Spring Boot · Python · Flask · Node.js · Express · MongoDB · Docker
